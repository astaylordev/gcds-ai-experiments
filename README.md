# GC Design System - React Skeleton App

A skeleton React application using the [Government of Canada Design System](https://design-system.alpha.canada.ca/) (GC Design System).

Built with [Vite](https://vite.dev/) + [React](https://react.dev/) + [@cdssnc/gcds-components-react](https://github.com/cds-snc/gcds-components).

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
npm run preview
```
